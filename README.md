# Lantra-Web

Lantra-Web is a web application designed to stream audio from a Lantra-Server. 

## Features

### Player
- Plays audio streamed from a Lantra-Server
- Player Controls (coming soon): control playback
- Media details display (coming soon): displays metadata such as media image, title, artist, etc.

### Poweramp Library (Coming soon)
- Specifically designed to work with PowerAmp, the library page will allow users to view and manage their music library

### Visualiser (Coming soon)
- the Visualiser page will allow users to select between visualiser presets while audio is playing
